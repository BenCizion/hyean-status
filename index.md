## ✅ 완료 — 지금 hyean.ai에서 동작합니다

**회원 / 프로필**
- 이메일 회원가입 · 로그인
- 가입 직후 온보딩: 닉네임(중복 실시간 확인) + 아바타 선택
- 마이페이지: 내 정보 · 내 글 · 저장한 캠페인

**포인트(냥)**
- 가입 시 500냥 즉시 지급
- 행동 기반 적립: 출석 · 글쓰기 · 댓글 · 공유 · 베팅 참여 (현금 충전·환전 없음)

**예측 캠페인**
- 캠페인(마켓) 생성 / 목록 / 상세
- 이진(YES·NO) · 다중선택 마켓, 냥 베팅
- 실시간 확률 + **확률 추이 시계열 그래프**
- 캠페인 저장(북마크), 공유(링크·트위터·카카오)
- 운영자 승인 · 결과 판정 · 정산(풀 비례 배분)

**커뮤니티**
- 트위터식 짧은 글(140자) 작성 → 메인 피드 노출
- 캠페인별 댓글

**인프라**
- 서울(Seoul) 리전 · hyean.ai 도메인 + 보안 인증서(SSL)
- 코드 변경 시 자동 배포 파이프라인

## 🧪 가볍게 체험해 보세요

1. https://hyean.ai 접속 → **회원가입**(이메일) → 닉네임·아바타 설정
2. **가입 보너스 500냥** 확인
3. **캠페인** 메뉴 → 마음에 드는 예측에 냥 **베팅** → 확률·그래프 변화 확인
4. **홈 피드**에 짧은 글 남기기 / 캠페인에 **댓글·공유** → 냥 적립 확인
5. (선택) 캠페인 **저장** → 마이페이지에서 다시 보기

> 지금은 기능 검증 단계라 디자인은 임시입니다. 톤앤매너는 차주 함께 정합니다.
